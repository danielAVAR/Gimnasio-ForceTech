# SISTEMA DE GESTIÓN PARA GIMNASIO FORCE-TECH :muscle:
#### _DESCRIPCION:_
El gimnasio ForceTech actualmente no cuenta con ningún sistema digital que le permita 
gestionar de una manera eficiente la información de sus clientes, servicios y operaciones internas. 
La mayoría de los procesos aún se realizan de forma manual o incluso sin una estructura centralizada, 
lo cual genera dificultades en el gestión, seguimiento y toma de decisiones.

Para ello, se creo el presente sistema el cual resuelve esta problemática, haciendo uso de tecnologías de almacenamiento de datos,
gestión de servicio, clientes y progreso. 

Este sistema fue elaborado por un grupo de programadores en desarrollo como parte de su formaciòn en el uso de tecnologías como GiHub, ClickUp, VisuaBasic, Git, entre otros,
y que servirá como parte de su evaluación de conocimientos por parte de su instructor.
